# Data Sources

The raw source files are not included due to size (total ~1.4 GB).

Download directly from NYC Open Data:
- [Crashes](https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Crashes/h9gi-nx95)
- [Vehicles](https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Vehicles/bm4k-52h4)
- [Persons](https://data.cityofnewyork.us/Public-Safety/Motor-Vehicle-Collisions-Person/f55k-p6yu)

The pre-processed analytical dataset (`final_processed_data.parquet`) is included
and covers 101,000 driver records across 24 columns — ready for analysis.
